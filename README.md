# Student-Management-System
This project is about managing student information like student details, courses, and enrollments using a web-based system.
## Student Management System

### Description
This project focuses on the conceptual design of a student management system for handling academic records.

### Technologies
Java, Spring Boot, React, REST API, MySQL / Oracle Database

### System Overview
The system is designed to manage student, course, and enrollment information through a web-based application.

### Key Functionalities
- Add, update, delete, and view student records
- Manage course details
- Handle student enrollments
- Backend structured using MVC architecture

### Scope
The project emphasizes understanding full-stack architecture, RESTful API design, and database interaction concepts.
